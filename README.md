**UIDAI Smart Enrolment Analytics1** is a full-stack web application designed to analyze Aadhaar enrolment and update requests across India.  
It provides **interactive dashboards, state/district-level insights, predictive analytics, and reporting features** to help UIDAI improve operational efficiency and decision-making.  

- **Frontend:** React.js  
- **Backend:** Spring Boot  
- **Database:** MySQL  
- **Charts & Visualization:** Recharts, Leaflet.js (Interactive maps)  
- **Styling:** Tailwind CSS
